# TwitCasting Recorder Py
A python script for recording TwitCasting live streams

## Usage
```python
$ python main_record_twitcast.py {USER_ID} {LOOP_TIME}
```
Sample:
```python
$ python main_record_twitcast.py user_id 60
```
